# ①課題名
金滅の刃ー無限くじ編ー

## ②課題内容（どんな作品か）
- とにかく沢山くじを引きたいと思っても、前回まではブラウザに全結果を記載したため、試行回数の増加が早い段階でレスポンスの低下につながっていました。
- そこで、ボタン⓶を新たに実装しました。ローカルストレージに一定回数の結果を保存することで、ブラウザへの負荷を最低限に沢山くじを引けるようにしました。結果の配列は、slice(-10)を活用し、10のみ常に保持し、古いデータは削除されて常に新しいデータに置き換わるようにしています。
- また、保有資産が減っていく感じが見えないと面白くないなと思ったので、ローカルストレージ最初に資産額を入れてゲームを始めるとデータが保存されて保有資産の金額をいじれないままくじを引き続ける他なくなるボタン⓷を実装しました。借金が10回続くとゲームオーバーになるギミックも入れました！
